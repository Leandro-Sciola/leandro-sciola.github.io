libraries

In this directory are the libraries of your application.

You can create your classes in this directory and call them anywhere in your application. Example:

File: Foo.php

<?php

namespace App\Lib;

class Foo
{
    public function bar()
    {
        return 1;
    }
}

Now just instantiate the class in the controller, model or other library. Example:

File: /layers/Controllers/MyController.php

<?php

namespace Layer\Controller;

use Framework\Controller;
use App\Lib\Foo;

class MyController extends Controller
{
    public function demo()
    {
        $Foo = new Foo;
        echo $Foo->bar(); // 1
    }
}

or

File: /layers/Models/MyModel.php

<?php

namespace Layer\Model;

use Framework\Model;
use App\Lib\Foo;

class MyModel extends Model
{
    public function demo()
    {
        $Foo = new Foo;
        echo $Foo->bar(); // 1
    }
}
